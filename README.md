# IoT-Based Wearable Epileptic Seizure Alert System

## Overview

This project is a wearable embedded system designed to assist in monitoring epileptic seizure events. The prototype continuously monitors physiological and motion-related parameters and can notify caregivers during emergencies.

The objective is to improve patient safety through continuous monitoring using embedded systems and IoT technologies.

> **Note:** Detailed implementation, firmware, and hardware design are not publicly available because the project contains intellectual property currently under protection.

---
## Features

- Wearable embedded device
- Real-time monitoring
- IoT connectivity
- GPS-based location sharing
- Emergency caregiver alerts
- Low-power embedded design

## Technologies

- Embedded C
- ESP32-C3
- Arduino IDE
- MPU6050
- GPS Module
- Blynk IoT
- Git & GitHub
